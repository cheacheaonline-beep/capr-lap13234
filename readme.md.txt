# 🔐 Cyber Security Lab Project

## 📌 Overview
This is a simple cybersecurity lab project created for learning and practicing web security concepts such as authentication systems, login handling, and basic vulnerability simulation in a controlled environment.

⚠️ This project is for educational purposes only.

---

## ⚙️ Features
- 🆕 User registration system
- 🔐 Login system (authentication)
- 📊 Dashboard for monitoring requests
- 🧪 Security testing environment
- 🧨 XSS simulation (for learning purposes)

---

## 🧠 Technologies Used
- HTML
- JavaScript
- Python (Flask)
- REST API
